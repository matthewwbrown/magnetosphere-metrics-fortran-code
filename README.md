# magnetosphere-metrics-fortran-code
Methods for computing the Integrated Slippage Activity and Unsigned Flux in y-z plane from input parameters. As well as a small Fortran program that calculates the resistive-Hall reconnection.
This page is based on the Magnetospheric Model (TA16) from N. A. Tsyganenko and V. A. Andreeve, modified by S.Stanish. The file recon_res-hall.f90 is modified from the recon.f90 file on [magnetosphere-T-models](https://github.com/sastanish/magnetosphere-T-models) from S. Stanish.
